# 二叉树遍历

## 前序

根->左->右

1. 递归

   ```java
       public void traversalWithRecursion(TreeNode root) {
           if (root == null) {
               return;
           }
           System.out.print(root.val + " ");
           traversalWithRecursion(root.left);
           traversalWithRecursion(root.right);
       }
   ```

2. 非递归

   ```java
    public void traversalWithoutRecursion(TreeNode root) {
           if (root == null) return;
           Stack<TreeNode> stack = new Stack<>();
           stack.push(root);
           while (!stack.empty()) {
               TreeNode curNode = stack.pop();
               System.out.print(curNode.val + " ");
               //手动入栈代替递归函数栈
               if (curNode.right != null) stack.push(curNode.right);
               if (curNode.left != null) stack.push(curNode.left);
           }
           System.out.println();
       }
   ```

   

   

## 中序

左->根->右

1. 递归

   ```java
    public void traversalWithRecursion(TreeNode root) {
           if (root == null) {
               return;
           }
           traversalWithRecursion(root.left);
           System.out.print(root.val + " ");
           traversalWithRecursion(root.right);
       }
   ```

   

2. 非递归

   ```java
    public void traversalWithoutRecursion(TreeNode root) {
           if (root == null)
               return;
           Stack<TreeNode> stack = new Stack<>();
           while (!stack.isEmpty() || root != null) {
               if (root != null) {
                   //traversalWithRecursion(root.left);
                   stack.push(root);
                   root = root.left;
               } else {
                   root = stack.pop();
                   System.out.print(root.val + " ");
                   root = root.right;
               }
           }
       }
   ```

   

## 后序

左->右->根

1. 递归

   ```java
       public void traversalWithRecursion(TreeNode root) {
           if (root == null) {
               return;
           }
           traversalWithRecursion(root.left);
           traversalWithRecursion(root.right);
           System.out.print(root.val + " ");
       }
   ```

   

2. 非递归

   ```java
    public void traversalWithoutRecursion(TreeNode root) {
           Stack<TreeNode> stack1 = new Stack<>();
           Stack<TreeNode> stack2 = new Stack<>();
   
           stack1.push(root);
   
           while (!stack1.empty()) {
               root = stack1.pop();
               stack2.push(root);
               if (root.left != null) {
                   stack1.push(root.left);
               }
               if (root.right != null) {
                   stack1.push(root.right);
               }
           }
   
           while (!stack2.empty()) {
               System.out.print(stack2.pop().val+" ");
           }
       }
   ```
